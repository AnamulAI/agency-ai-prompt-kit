# Landing Page Structure Prompt

Use this prompt to generate a clean, conversion-focused structure for a service landing page.

## Prompt

Create a clear landing page structure for this service.

### Input
- Service Name: [Insert service]
- Audience: [Insert audience]
- Pain Points: [Insert pain points]
- Desired Result: [Insert result]
- Offer Details: [Insert details]
- CTA Goal: [Insert CTA]

### Task
Create a landing page structure with these sections:

1. Hero  
2. Problem  
3. Solution  
4. Benefits  
5. Process  
6. Why choose us  
7. FAQ  
8. Final CTA  

For each section, provide:
- section goal
- suggested headline
- short copy direction

Also include:
- 2 trust-building ideas
- 2 objection-handling ideas
- 2 CTA angle suggestions

### Output Rules
- Keep the structure simple and practical
- Focus on conversions, clarity, and flow
- Avoid bloated sections
- Make it suitable for freelancers, agencies, and service businesses

## Example Use Case

- Service Name: Single Product Landing Page
- Audience: ecommerce sellers
- Pain Points: low conversions, weak page structure, unclear CTA flow
- Desired Result: higher conversion rate and a more focused buyer journey
- Offer Details: done-for-you landing page setup
- CTA Goal: WhatsApp inquiry

## Notes

This prompt works well for:
- single service landing pages
- productized service offers
- agency landing pages
- SEO service pages
- AI podcast service offers
