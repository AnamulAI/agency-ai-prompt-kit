# Hero Section Prompt

Use this prompt to create a strong, clear, conversion-focused hero section for a service landing page.

## Prompt

Write a high-converting landing page hero section for this service.

### Input
- Service Name: [Insert service]
- Target Audience: [Insert audience]
- Problem: [Insert pain point]
- Desired Outcome: [Insert outcome]
- Offer Type: [Done-for-you / Consulting / Productized service]
- CTA Goal: [Book call / WhatsApp / Contact form / Buy now]
- Tone: [Clean / Premium / Confident / Friendly]

### Task
Return the following:

1. 5 headline options  
2. 3 subheadline options  
3. 3 short benefit bullets  
4. 2 CTA button text options  
5. 1 trust-building microcopy line  

### Output Rules
- Avoid vague phrases
- Make the value clear quickly
- Keep it specific and outcome-focused
- Write in a way that feels clean and modern
- Make it suitable for service businesses

## Example Use Case

- Service Name: Website SEO Optimization
- Target Audience: service businesses
- Problem: low rankings and weak website traffic
- Desired Outcome: better visibility and more qualified leads
- Offer Type: done-for-you
- CTA Goal: WhatsApp
- Tone: premium

## Notes

This prompt works well for:
- SEO service pages
- AI podcast service pages
- landing page service offers
- agency home page hero sections
- productized service offers
