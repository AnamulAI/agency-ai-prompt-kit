# Follow-up Outreach Prompt

Use this prompt to create short follow-up messages for outreach without sounding pushy.

## Prompt

Write 5 short follow-up messages for a cold outreach sequence.

### Input
- Service: [Insert service]
- Prospect Type: [Insert type]
- Original Value Proposition: [Insert angle]
- Tone: [Friendly / Direct / Casual / Professional]

### Task
Create 5 follow-up messages that are:

- respectful
- short
- value-focused
- non-pushy
- easy to reply to

### Output Rules
- Do not sound desperate
- Do not repeat the same line in different ways
- Keep each follow-up clearly different
- Make the wording natural and conversational
- Include a soft CTA where relevant

## Example Use Case

- Service: AI Podcast for Local Business
- Prospect Type: local business owner
- Original Value Proposition: done-for-you short podcast content that helps build trust and stay visible
- Tone: friendly

## Notes

This prompt works well for:
- freelancer outreach sequences
- agency follow-up messages
- email follow-ups
- DM follow-ups
- service-based client acquisition
