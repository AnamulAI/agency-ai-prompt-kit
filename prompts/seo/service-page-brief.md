# SEO Service Page Brief Prompt

Use this prompt to build a practical SEO-focused service page brief for a local or online service business.

## Prompt

Create an SEO-friendly service page brief.

### Input
- Service Name: [Insert service]
- Location: [Insert location if needed]
- Target Customer: [Insert audience]
- Core Problem: [Insert problem]
- Search Intent: [Insert intent]

### Task
Create a service page brief with:

1. 5 primary keyword ideas  
2. 8 secondary keyword ideas  
3. one strong page angle  
4. 3 H1 ideas  
5. recommended page sections  
6. 5 FAQ ideas  
7. internal linking suggestions  
8. one conversion angle  
9. one trust-building section idea  
10. one local SEO angle if relevant  

### Output Rules
- Keep it relevant for service businesses
- Avoid keyword stuffing
- Focus on search intent and conversion intent together
- Make the structure practical and usable

## Example Use Case

- Service Name: Website SEO Optimization
- Location: Chattogram
- Target Customer: service business owners
- Core Problem: website is not ranking well and is not generating enough leads
- Search Intent: looking for SEO help for a business website

## Notes

This prompt works well for:
- SEO service pages
- local SEO pages
- service-based business websites
- lead generation service pages
